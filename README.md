# 2nd-project
### 👋&nbsp; 1.소개
   - 비트캠프 2차 프로젝트 진행(2021-10-07 ~ 2021-10-20)
   - 프로젝트 명 : Hello World
   - 프로젝트 인원 : 5명
   - 프로젝트 선정이유 : 지금은 사라져버린 싸이월드의 감성을 추억하고 떠올리기 위해 미니홈피 사이트를 기획
   - 역할 : 로그인기능, 로그인화면UI, 방문자 수(today,total), 프로필 조회, 수정 <br>
